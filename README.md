# jellyfin-web-docker

A docker image of the [jellyfin-web](https://github.com/jellyfin/jellyfin-web) repository.

# Images

A build will be triggered daily. The following Jellyfin version will be built

- 10.10.x
- 10.11.x
- master

The following Jellfin web versions are deprecated and will not be updated (You can still find them in the [package](https://github.com/LASER-Yi/jellyfin-web-docker/pkgs/container/jellyfin-web))

- 10.7.x
- 10.8.x
- 10.9.x
